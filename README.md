* `v3.10`
* `v4.0`
* `v4.1`
* `v4.2`

Doesn't work on ruTorrent:
* `v4.3`

## Install

```bash
$ cd /var/www/rutorrent/plugins/theme/themes
$ git clone https://github.com/djerfy/rutorrent-materialdesign.git MaterialDesign
$ chown -R www-data:www-data MaterialDesign
```
